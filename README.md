# Matching Dating App

MDA is a Web application for FPT University student.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install.

```bash
npm i
npm run dev #Start development server
```

## Tech Stack

```python
Vite Webpack for ReactJS

# UI
TailwindCSS

# Other
React-router for routing
Theme Context

```
